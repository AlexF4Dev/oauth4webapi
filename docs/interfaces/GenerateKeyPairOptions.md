## Properties

### extractable

• `Optional` **extractable**: `boolean`

Indicates whether or not the private key may be exported.
Default is `false`.

___

### modulusLength

• `Optional` **modulusLength**: `number`

(RSA algorithms only) The length, in bits, of the RSA modulus.
Default is `2048`.
